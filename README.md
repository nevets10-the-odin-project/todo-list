# todo-list
TOP Project: Todo Lis

## Outline
Todo items need:
- Title
- Description
- Due Date
- Priority
- Notes (Optional)
- Checklist (Optional)

Functionality:
- Separate lists of todo items/projects
- - There should be a default list

UI Features:
- View all projects
- View all todos in each project (Just basic info like title and due date)
- Expand a single todo view/edit it
- Delete a todo
